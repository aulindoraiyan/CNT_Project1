Welcome to our first project for Net-centric Computing. This Python project is a simplified FTP client that works in passive mode. It includes the following commands:

- ls
- cd
- get
- put
- delete
- quit

To run: Using the Windows/Linux/MacOS built-in command line FTP client, run (without quotations): "python myftp.py inet.cs.fiu.edu"
